            Dan Wahlin Mod batman 3 months ago

            The new version of Angular definitely has more setup required but that's made quite easy by using the Angular CLI. The reason is that Angular is a full framework (vs. just a library) that leverages the latest technologies and not all of them are supported directly in browsers yet. But, with the project setup provided by the CLI you can leverage cutting-edge features today. You can get more details here on getting started.

            https://angular.io/guide/qu...

            Having said that, if you're building a small app and don't need the full framework features provided by Angular then check out something like Vue.js (https://vuejs.org/). While it isn't truly a "framework" (they like to say it is....but it really isn't) it provides a simple way to build Single Page Apps without a lot of setup work.
